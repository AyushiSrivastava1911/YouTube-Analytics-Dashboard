# YouTube-Analytics-Dashboard
Interactive YouTube Analytics Dashboard built using Power BI. It provides insights into video performance, audience engagement, and channel growth through data visualization.

# Key Performance Trends:
• Short views account for 85.7% of total views while long videos contribute only 14.3% 
  indicating that viewers strongly prefer short-form content. 
• Overall engagement rate is 0.44% indicating limited audience interaction. 
• Views decrease from 303142 in July to 37167 in October, showing a drop in audience 
  reach. 
• Watch time is more in July and lowest in October. 

# Interesting Patterns: 
Short videos not only get most views but also tend to have higher Average CTR, showing  
stronger audience engagement. 
Average CTR shows slight month-to-month fluctuations. 
Subscriber growth is higher for shorts than long videos showing that people like 
short content more. 
The top 5 videos got most of the views, showing that a few videos are very popular. 

# Recommendations: 
1. Since Shorts are getting more views so the channel should focus on creating more 
   short-form content to improve reach and retention. 
2. Average CTR is fluctuating month to month. Improving thumbnails and titles can  
   attract more clicks and improve overall performance. 
3. The top 5 videos have most views so focusing on similar content can help increase 
   reach and bottom 5 videos have lowest views so reviewing this can help identify 
   what doesn’t work so similar content can be avoided in the future.
